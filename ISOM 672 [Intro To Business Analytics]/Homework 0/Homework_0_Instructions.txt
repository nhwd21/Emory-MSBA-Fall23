Optional Homework on Python Fundamentals

The primary objective of this homework is to help you become acquainted with the Python knowledge necessary for this course. Before progressing further into the course assignments, you should be proficient in addressing such queries using Python. Submission of this optional homework will not impact your grade.

Dataset: foodorders.tsv

Links to an external site.

Tasks

    Load the data into a suitable structure.
    Display the first five records from the dataset.
    Report the number of rows and columns present in the dataset.
    Identify the most frequently ordered item and state its order count.
    Calculate the total revenue generated by the firm.
    Determine the average amount spent per order.
    Count the number of unique items ordered from the dataset.
